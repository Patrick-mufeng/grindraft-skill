# Design Principles · 封面设计通用规范

## 核心哲学

封面设计的唯一目标：**3 秒内让读者理解文章是什么、值不值得点开**。

AI 不是随机拼凑元素，而是理解文章后做出的设计决策。每一条规则背后都有理由。

---

## 一、分析维度

拿到文章后，AI 必须从以下三个维度分析，缺一不可：

### 1. 情绪调性

| 调性 | 特征 | 配色偏向 | 字体偏向 | 典型模板 |
|------|------|----------|----------|----------|
| 冷峻 | 理性、克制、专业 | 蓝/灰/黑、低饱和 | 无衬线、等宽 | 瑞士极简、蓝图、全息未来 |
| 温暖 | 治愈、生活、人情 | 暖橙/粉/米、中高饱和 | 圆体、手写 | 柔光暖调、水彩渲染、植物标本 |
| 激昂 | 力量、震撼、冲击 | 红/金/高对比 | 粗衬线、黑体 | 暗金奢华、构成主义、社论杂志 |
| 轻松 | 幽默、亲和、年轻 | 多色/粉彩/明亮 | 圆体、趣味体 | 孟菲斯、像素艺术、迈阿密热浪 |
| 严肃 | 权威、深度、分析 | 黑白/深蓝/暗金 | 衬线、细黑 | 社论杂志、维多利亚、装饰艺术 |

### 2. 内容领域

每个领域有默认的视觉语言：

- **科技**：冷色 + 几何 + 无衬线 + 数据感
- **商业**：暗色 + 金色 + 衬线 + 信任感
- **生活**：暖色 + 柔和 + 圆体 + 亲近感
- **教育**：清晰 + 留白 + 易读 + 结构感
- **娱乐**：高饱和 + 动感 + 趣味 + 冲击力
- **人文**：古朴 + 纹理 + 质感 + 沉淀感

### 3. 个人 IP

如果文章来自个人创作者而非企业号：
- **IP 名称/Logo** 必须出现在封面固定位置
- 色调可围绕 IP 品牌色微调
- 字体风格应与 IP 人设一致（比如"硬核程序员"→等宽，"温柔生活博主"→手写）

## 二、色彩规则

### 配色三原则

1. **60-30-10 法则**：主色 60%（背景）+ 辅色 30%（卡片/区块）+ 强调色 10%（按钮/高亮）
2. **对比度底线**：正文与背景对比度 ≥ 4.5:1（WCAG AA），标题 ≥ 3:1
3. **不超过 4 色**：主色 + 辅色 + 强调色 + 文本色，额外颜色仅用于微装饰

### 色板定义

每个模板必须提供完整色板：

```yaml
palette:
  bg: "#0a0a14"        # 背景色
  surface: "#141428"   # 卡片/区块色
  primary: "#00f0ff"   # 主色（标题、重点元素）
  accent: "#ff00aa"    # 强调色（点缀、高亮）
  text: "#e0f0ff"      # 正文颜色
  muted: "#6a7a9a"     # 次要文字
```

### 禁止的配色

- ❌ 纯黑 `#000` + 纯白 `#fff` 大面积对比（刺眼）
- ❌ 红绿搭配（色盲不友好）
- ❌ 荧光色大面积做背景（视觉疲劳）
- ❌ 超过 5 种不同色相混用

## 三、字体规则

### 字体层级

```
一级标题（H1）：最大、最粗、品牌感最强
二级标题（H2）：中等大小，引导阅读
正文/描述 ：易读、行高 ≥ 1.5
标签/徽章 ：小号、紧凑、全大写或等宽
```

### 字体搭配原则

1. **不超过 2 种字体族**：一个用于标题（表现风格），一个用于正文（保证易读）
2. **中英混排**：中文优先 `PingFang SC` / `Noto Sans SC`，英文优先 `Inter` / `SF Pro`
3. **衬线 vs 无衬线**：
   - 衬线（serif）：权威、传统、优雅 → 商业/人文/品牌
   - 无衬线（sans-serif）：现代、干净、高效 → 科技/教育/工具

### 字体大小阶梯（2.35:1 基准）

| 层级 | 推荐大小 | 用途 |
|------|---------|------|
| H1 | 48-72px | 封面主标题 |
| H2 | 24-36px | 副标题/引言 |
| Body | 14-18px | 描述文字 |
| Caption | 11-13px | 标签/徽章/版权 |

⚠️ **禁止在封面卡片 CSS 中使用 `vw` 单位**。封面卡片的宽度由 Grid 列宽决定而非视口宽度，`clamp(30px, 4.5vw, 64px)` 在 1920px 屏幕上会算出 86px，远超卡片实际空间，导致内容溢出被 `overflow:hidden` 裁掉。所有字号使用固定 `px` 值或相对父容器的 `%`。

## 四、布局规则

### 2.35:1 封面布局范式

```
┌─────────────────────────────────────────────┐
│ [徽章行]                          [右上角标] │
│                                              │
│  ┌──────────────────┐                        │
│  │                  │                        │
│  │   主标题区       │      [装饰元素]        │
│  │   (占 50-70%)   │                        │
│  │                  │                        │
│  └──────────────────┘                        │
│                                              │
│  描述文字 (≤540px宽)                         │
│                                              │
│  [特性标签] [标签] [标签]                    │
│                                     [品牌签] │
└─────────────────────────────────────────────┘
```

### 1:1 封面布局范式（简化版）

```
┌─────────────────────┐
│                     │
│     [品牌/标识]     │
│                     │
│   ┌─────────────┐   │
│   │  主标题     │   │
│   │  (居中)     │   │
│   └─────────────┘   │
│                     │
│   AI 智能排版引擎   │
│                     │
│ [15套主题 · 开源]   │
│                     │
└─────────────────────┘
```

### 布局禁止项

- ❌ 主标题超出可视区域 70%
- ❌ 文字与装饰元素重叠导致不可读
- ❌ 信息密度过高（超过 5 个独立信息组）
- ❌ 标题被裁切或换行过多（≤3 行）

## 五、装饰元素规则

### 装饰的目的

装饰不是填充空白，而是**强化调性、引导视线**。

| 调性 | 推荐装饰 |
|------|---------|
| 科技 | 网格线、数据点、扫描线、发光点 |
| 商业 | 细金线、几何边框、对称花纹 |
| 生活 | 柔光球、有机形状、叶片剪影 |
| 严肃 | 极细分割线，少量几何装饰 |
| 轻松 | 波点、几何形、趣味图标 |

### 装饰密度

- 极简风格：≤1 种装饰
- 常规风格：2-3 种装饰
- 繁复风格（维多利亚/洛可可）：4-5 种装饰，但必须控制不抢文字

---

## 六、1:1 适配规则

生成封面时必须同时输出 1:1 版本，规则如下：

1. **信息精简**：1:1 只保留 Logo/标识 + 标题 + 一句话 + 最关键的 1-2 个标签
2. **标题放大**：1:1 的标题字号可比 2.35:1 大 10-20%，因为空间更集中
3. **布局居中**：1:1 默认居中布局，除非模板指定了其他方式
4. **配色一致**：与 2.35:1 使用完全相同的色板
5. **装饰简化**：1:1 的装饰元素数量减半

## 七、下载规格

| 档位 | 分辨率（2.35:1） | 分辨率（1:1） | 用途 |
|------|-----------------|--------------|------|
| 4K | ~3840×1634 | ~3840×3840 | 打印/高清展示 |
| 2K | ~2560×1089 | ~2560×2560 | 电脑端公众号 |
| 1080p | ~1920×817 | ~1920×1920 | 移动端/快速预览 |
| SVG | 矢量 | 矢量 | 可编辑/缩放 |

### 文件命名

```
WeMD_{模板英文名}_{档位}_{时间戳}.{格式}

示例：
WeMD_CyberNeon_4K_20260602-143022.png
WeMD_SwissMinimal_1-1_SVG_20260602.svg
```

## 八、质量检查清单

生成封面后必须自查：

- [ ] 主标题在 3 秒内可读完（≤15 字）
- [ ] 文字与背景对比度达标
- [ ] 2.35:1 + 1:1 双尺寸均已生成
- [ ] 色板来自指定模板（未混用其他模板色）
- [ ] 装饰元素未遮挡文字
- [ ] 4K/2K/1080p/SVG 下载按钮均可用
- [ ] 个人 IP（如有）出现在封面
- [ ] 文件命名符合规范
