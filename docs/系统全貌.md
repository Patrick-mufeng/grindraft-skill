## 双模式切换

```
用户初始化 ──▶ cold-start (前 5 篇)
                  │
                  │ 每写一篇 prediction → cold_start_remaining -1
                  │
                  ▼ cold_start_remaining = 0
             calibration (完整预测)
                  │
                  │ 每次复盘 → calibration_samples +1
                  │ 连续 3 次同向偏差 → 提示 bump
                  ▼
             rubric 升级 (v0 → v1 → v2...)
```

## 每步产生什么文件

```
Step 1 · trends
  candidates.md  ← 新增候选（粗打分 + 去重）

Step 2 · seed
  candidates.md  ← 候选标 tier1 + deep_read

Step 3 · write
  articles/{标题}_{日期}/
  └── draft.md
    ├── frontmatter: title (占位), summary (占位)
    ├── 📝 标题候选 (4-5个)
    ├── 📋 简介 (一句 + 一段)
    └── 正文
  candidates.md  ← 选中候选 → status: drafting

Step 4 · humanize（可选 — 只有用户说"去 AI 味"时才执行）
  articles/{标题}_{日期}/final.md  ← 四层自检修完的终稿
  plates/style-diffs/{标题}_{日期}_before.md  ← AI 初稿备份
  style_guide.md    ← 自动沉淀改稿 pattern

  如果跳过 humanize，用户手动改 draft.md 后直接保存为 final.md 即可。

Step 5 · illustrate（可选）
  articles/{标题}_{日期}/illustrations/
  ├── shot-list.md       ← 配镜表
  └── 01-{主题}.png      ← 实际配图（API 生成）

Step 6 · cover
  articles/{标题}_{日期}/cover/
  ├── preview.html       ← 双图预览（2.35:1 + 1:1）
  ├── cover-2x35.png     ← 公众号首图
  └── cover-1x1.png      ← 圆角小图

Step 7 · format
  articles/{标题}_{日期}/
  ├── output.html         ← 清洁版（直接粘贴公众号）
  └── output-preview.html ← 预览版（手机模拟 + 复制按钮）

Step 8 · predict
  articles/{标题}_{日期}/prediction.md  ← 7维分 + bucket + 概率 (immutable)
  .grindraft-state.json  ← cold_start_remaining -1
  candidates.md          ← status: published

Step 9 · publish
  prediction.md header  ← 写入 URL + 发布时间

Step 10 · retro (T+3天后)
  articles/{标题}_{日期}/retro/report.md  ← 原始数据
  prediction.md          ← ## 复盘 段追加
  rubric_notes.md        ← 新观察记录
  style_guide.md         ← 新写作 pattern
```

## 7 维评分体系

```
composite = (HK×2 + NR×2 + QA×1.5 + UT×1.5 + DT + EP + SC×0.5) / 9.5 × 2.0
范围：0-10（与 cheat-on-content 同量纲）

HK 标题钩力     → 阅读量       |  NR 叙事牵引力 → 完读率
QA 金句锚点     → 划线/在看    |  UT 实用密度   → 收藏率
DT 思辨激发度   → 留言率       |  EP 情绪峰值   → 分享率
SC 结构闭环     → 完整感(软)   |
```
