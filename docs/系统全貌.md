## 双模式切换

```
用户初始化 ──▶ cold-start (前 5 篇)
                  │
                  │ 每写一篇 prediction → cold_start_remaining -1
                  │
                  ▼ cold_start_remaining = 0
             calibration (完整预测)
                  │
                  │ 每次复盘 → calibration_samples +1
                  │ 连续 3 次同向偏差 → 提示 bump
                  ▼
             rubric 升级 (v0 → v1 → v2...)
```

## 每步产生什么文件

```
Step 1 · trends
  candidates.md  ← 新增候选（粗打分 + 去重）

Step 2 · seed
  candidates.md  ← 候选标 tier1 + deep_read

Step 3 · write
  drafts/2026-05-28_<中文标题>.md
    ├── frontmatter: title (占位), summary (占位)
    ├── 📝 标题候选 (4-5个)
    ├── 📋 简介 (一句 + 一段)
    ├── 🎨 封面提示词 (2-3套)
    └── 正文
  candidates.md  ← 选中候选 → status: drafting

Step 4 · humanize
  drafts/<标题>.md  ← 覆盖（四层自检修完）
  style_guide.md    ← 自动沉淀改稿 pattern

Step 5 · format
  output/<标题>.html  ← AI 自动设计主题，可直接粘贴公众号后台

Step 6 · predict
  predictions/<标题>.md  ← 7维打分 + 方向押注 (immutable)
  .grindraft-state.json  ← cold_start_remaining -1
  candidates.md          ← status: published

Step 7 · retro (T+3天后)
  predictions/<标题>.md  ← ## 复盘 段追加
  rubric_notes.md        ← 新观察记录
```

## 每条 Skill 干什么、何时触发

| Skill | 一句话 | 触发词 |
|---|---|---|
| init | 首次使用建项目骨架，判定 cold-start/calibration | "磨稿初始化" |
| trends | 从 aihot 抓 AI 圈热点，去重粗打分入选题池 | "抓热点" |
| seed | 围绕你的经历深挖选题角度，收敛到一句话立意 | "找选题" |
| write | AI 写初稿，自动附带标题候选+简介+封面提示词 | "写文章" |
| humanize | 四层去 AI 味（L1-L4），自动沉淀你的改稿 pattern 到 style_guide | "去 AI 味" |
| format | AI 自动设计主题，Markdown → 公众号 HTML，一键排版 | "排版" |
| predict | 7 维盲打分 + 发布登记，cold-start 自动递减 | "启动预测" |
| retro | 粘贴 3 个核心数据 → 派生比率 → 逐假设验证 → 跨样本检测 | "复盘" |
| status | 显示模式/rubric/草稿进度/待复盘/选题池 | "状态" |
| recommend | 按 composite 排序推荐下一篇选题 | "推荐选题" |
| persona | 从复盘评论数据派生读者画像 | "我的读者是谁" |
| bump | rubric 升级：全量重打 + 跨模型审核 | "升级 rubric" |

## 7 维评分体系

```
composite = (HK×2 + NR×2 + QA×1.5 + UT×1.5 + DT + EP + SC×0.5) / 9.5 × 2.0
范围：0-10（与 cheat-on-content 同量纲）

HK 标题钩力     → 阅读量       |  NR 叙事牵引力 → 完读率
QA 金句锚点     → 划线/在看    |  UT 实用密度   → 收藏率
DT 思辨激发度   → 留言率       |  EP 情绪峰值   → 分享率
SC 结构闭环     → 完整感(软)   |
```
