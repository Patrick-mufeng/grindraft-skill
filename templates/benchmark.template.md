## 我的基准指标

| 指标 | 基准值 | 计算方式 | 可用？ |
|------|--------|---------|--------|
| 平均阅读量 | {N} | 中位数 | ✅ |
| 分享率 | {X.X}% 或 — | 分享数中位数 / 阅读量中位数 | {'✅' if data else '—'} |
| 收藏率 | {X.X}% 或 — | 收藏数中位数 / 阅读量中位数 | {'✅' if data else '—'} |
| 留言率 | {X.X}% 或 — | 留言数中位数 / 阅读量中位数 | {'✅' if data else '—'} |
| 在看率 | {X.X}% 或 — | 在看数中位数 / 阅读量中位数 | {'✅' if data else '—'} |

> 标注"—"的指标因历史数据中缺少对应数字无法计算。这些基准值会随着后续新文章的复盘逐步积累。
