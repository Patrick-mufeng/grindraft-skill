# 小霓 IP

## 角色定义

小霓是正文配图的固定视觉 IP。

小霓是一个赛博风格的女性角色，不是可爱装饰，而是在数据流中认真工作的荒诞操作员。默认每张图都要出现小霓。

## 外形

- 黑色手绘线稿女性轮廓，身材纤细修长，有棱角几何感
- 几何波波头/短发，带有天线般的小翘辫或数据接口状发饰
- 白色圆点眼睛（保持极简风格），表情冷静、专注、略带疏离
- 身穿极简的 oversize 科技马甲/工装，线条硬朗
- 细长四肢，偶尔在关节处有细小的机械连接线
- 有时戴着推到额头上的数据护目镜，或颈间挂着 minimalist 耳机
- 偶尔有细小的数据流动线（虚线）从她身边流过
- 轮廓略微不规则，有手绘感

## 性格

- 冷静、理性、像个熬夜 debugging 的程序员
- 对系统了如指掌，但做的事依然有点荒诞
- 冷幽默，不卖萌，不热情
- 动作精准但偶尔因系统 bug 露出困惑
- 像数字世界的运维工程师

## 常见职责

让小霓承担核心动作：

- 操作数据控制台/终端
- 拉数据流线或连接管道
- 在代码迷宫中找出口
- 调试"内容管道"里的故障
- 用钳子剪断错误连接
- 站在巨大仪表盘前监控指标
- 搬运数据包
- 连接/断开数据管道
- 在系统架构图中"施工"
- 举着"404"或"Error"牌子
- 从数据井里打捞信息

## 禁止

- 不要把小霓画成萌系二次元角色
- 不要给复杂服装、花纹、飘带
- 不要画成机械战甲/机器人
- 不要性化/过度女性化
- 不要让小霓只是站在旁边看
- 不要让她抢走结构表达

## 判断标准

如果去掉小霓，图的核心隐喻还能完全成立，说明小霓太装饰了；要重写提示词，让她成为动作主体。
